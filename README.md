#MapPoint
此專案是為了試用 Parse 服務而開發 <br>
為以下報告題目的 Demo <br><br>
**主題** 快速建構MVP的利器 - Parse, Cloud Code 介紹 <br>
**時間** 2015/5/19 晚上7:30 <br>
**地點** 好想工作室 <br>
**工具** Parse, Bootstrap, jQuery

專案構想是自製類似 google map 我的地圖 服務 <br>
寫的很倉促，還沒有完整，目前提供以下功能
* 使用者 登入/註冊/忘記密碼
* 使用者建立新地圖
* 地圖上增加地點資訊

待辦
* 地點呈現在 map 上 (還沒有空接上 google map api …)
